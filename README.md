1. “能用，但别嫌弃”

   - 但如果你拉屎发现没纸，别找我们！（即：不提供技术支持）
2. “署名就行，别乱改”
   - 必须保留“IJUST-VPTL授权”的标签（就像公厕墙上必须写“来也匆匆，去也冲冲”）。
   - 禁止商用（除非你是冥界天地银行，专门印冥币买厕纸）。

   - 如果有人在你修改的代码上写小广告、掉手机、甚至掉进坑里——IJUST概不负责！
